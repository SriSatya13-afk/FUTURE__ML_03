# Task 3 - Resume / Candidate Screening System
## Future Interns - Machine Learning Internship

---

## 📌 Overview
Built an ML system that automatically screens and ranks 
resumes based on a given job role using NLP techniques.
This mirrors how real resume screening tools work in 
HR-tech companies and recruitment platforms.

---

## 🎯 What It Does
- Reads and processes resume text data
- Extracts skills and keywords from resumes
- Compares resumes with a job description
- Ranks candidates by match score
- Highlights missing skills per candidate
- Generates final hiring recommendation report

---

## 🛠️ Tools Used
| Tool | Purpose |
|------|---------|
| Python | Core language |
| spaCy | Skill extraction |
| NLTK | Text preprocessing |
| Scikit-learn | TF-IDF + Cosine Similarity |
| Matplotlib & Seaborn | Charts and heatmaps |
